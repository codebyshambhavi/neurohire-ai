# NeuroHire AI — ML Engine

This directory will contain the machine learning modules that power NeuroHire AI's multimodal interview intelligence.

Planned components:

- **AnswerMind** — NLP engine for answer quality, relevance, and semantic analysis
- **SpeechIQ** — Audio engine for speech clarity, pace, tone, and vocal confidence
- **VisionNet** — Computer vision engine for engagement, eye contact, and non-verbal cues

Implementation is not yet started.
